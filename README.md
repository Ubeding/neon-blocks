# Neon Blocks

Juego gratuito de bloques para navegador, inspirado en los grandes clásicos de puzles.

## Jugar

Abre la versión publicada en GitHub Pages: `https://ubeding.github.io/neon-blocks/`

## Controles

- Flechas izquierda y derecha: mover.
- Flecha abajo: bajar.
- Flecha arriba o X: girar.
- Espacio: caída rápida.
- P: pausa.
- En móvil se muestran controles táctiles.

## Características

- Siete tipos de piezas.
- Puntuación, líneas, niveles y dificultad progresiva.
- Récord guardado en el navegador.
- Diseño adaptable a ordenador y móvil.
- Sin anuncios, cuentas ni recogida de datos.

## Ejecutar localmente

No necesita instalación. Descarga el repositorio y abre `index.html` en un navegador.
